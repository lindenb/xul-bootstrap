# xul-bootstrap
Bootstrap XUL application


# Running :

```bash
$ xulrunner --app hello01/application.ini  -jsconsole  -purgecaches 
```

or from firefox

```bash
firefox -app hello01/application.ini -jsconsole  -purgecaches
```
